# Домашнее задание к занятию «Продвинутые методы работы с Terraform»

## Содержание
- `src/` - исходный код Terraform
- `screenshots/` - скриншоты выполнения задания
- `README.md` - описание задания

## Выполненные задания:
1. Использование модулей и remote state
2. Создание локального модуля VPC
3. Операции с state (удаление, импорт)
4. Расширенный модуль VPC с поддержкой нескольких зон
5. Модуль Managed MySQL кластера
6. Создание S3 бакета
7. Интеграция с HashiCorp Vault
8. Разделение root модуля на независимые модули

## Структура проекта:
04/src/
├── main.tf # Основная конфигурация
├── variables.tf # Переменные
├── outputs.tf # Выводы
├── terraform.tfvars # Значения переменных (в .gitignore)
├── modules/ # Локальные модули
│ ├── vpc/ # Модуль VPC
│ └── marketing_vm/ # Модуль виртуальной машины
